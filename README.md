# zma

Z80/R800クロスアセンブラです。
Windowsのコマンドラインアプリですが、Linux/MacOS用にビルドすることも可能です。

Z80/R800 cross assembler.
It is a command line application for Windows, but can also be built for Linux.

## how to build

### for Linux and macOS

```sh
cd src/linux
make
```
